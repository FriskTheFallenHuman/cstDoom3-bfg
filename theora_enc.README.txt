==============================================================================
[ABOUT]

The "Theora-TH Encoder Source Code" corresponds to the source code of the
Theora-TH video encoder provided with CstDoom3-BFG (TH is the name of a custom
container for Theora videos) (last updated: April 30, 2024). The included
project files are for Visual Studio 2019 along with the Windows SDK
10.0.16299.0 (this SDK can be obtained using the Visual Studio installer).
